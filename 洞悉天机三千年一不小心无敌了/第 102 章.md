# 第 102 章 最后的赌局

我没睡着。

趴在那张破课桌上，眼睛闭着，脑子却他妈转得跟陀螺一样。

六十九小时。

选项 A、B、C。

操。

我翻了个身，盯着天花板。

天花板上有一块污渍，形状像……人脸？

"别他妈瞎想了。"我对自己说。

没用。

脑子根本停不下来。

---

"所以，你到底在纠结什么？"

一个声音忽然响起。

我猛地坐起来。

教室里没人。

"谁？"

"低头。"

我低头。

课桌上，不知何时多了一枚……棋子？

不，不是棋子。

是一块黑色的石头。

表面光滑，泛着微光。

"韩渊？"我试探性地喊了一声。

"不是。"石头发出了声音，"我是……系统。"

我："……"

"你是系统？"我盯着那块石头，"苏小棠创造的那个？"

"可以这么说。"石头说，"但我不是苏小棠。我是系统本身。"

"系统本身？"我笑了，"你还挺会给自己加戏的。"

"这不是戏。"石头的声音很平静，"韩林，你知道为什么会有选项 C 吗？"

我沉默。

"因为苏小棠给不了你真正的选择。"石头说，"A 和 B，都是她设计好的。无论你选哪个，都在她的计算之中。"

"那 C 呢？"

"C……"石头顿了顿，"是唯一的变数。"

"什么意思？"

"意思是，"石头说，"选 C，你会跳出这个棋局。"

我心头一跳。

"怎么跳？"

"我不知道。"石头说，"因为选 C 的人，从来没有过。"

"三千年，九十七代人，没人选过 C？"

"没有。"石头说，"他们都选了 A 或 B。"

"然后呢？"

"然后……"石头的声音变得有些飘忽，"然后他们都失败了。"

我浑身一冷。

"失败是什么意思？"

"死亡。"石头说，"或者……比死亡更糟。"

我没说话。

"韩林，"石头继续说，"苏小棠在等你选 A。她希望你理解她，同情她，然后……送她最后一程。"

"这有什么问题？"

"问题在于，"石头说，"如果她真的想死，三千年前就可以死了。"

我愣住。

"不死是诅咒，没错。"石头说，"但诅咒……是可以转移的。"

"转移？"

"对。"石头说，"苏小棠不需要死。她只需要……找到一个替代品。"

我背后的汗毛竖起来了。

"你是说……"

"选 A，你会成为下一个苏小棠。"石头说，"永生不死，永远孤独，永远……等待下一个能理解你的人。"

"操！"我忍不住爆了粗口，"你他妈早说啊！"

"苏小棠不会告诉你。"石头说，"她甚至……可能自己都不知道。"

"什么意思？"

"这个秘密，被藏在了系统的最深处。"石头说，"连苏小棠自己，都被蒙在鼓里。"

我沉默了很久。

"那 B 呢？"我问，"选 B 会怎样？"

"选 B，系统自毁。"石头说，"苏小棠继续活着，韩家的诅咒继续，三千年后，下一个九十八代出现，重复这个循环。"

"那 C 呢？"

"C……"石头说，"我不知道。"

"你不是系统吗？"

"我是系统的一部分。"石头说，"但 C 选项……不在我的认知范围内。"

"那是谁设计的？"

"韩家的祖先。"石头说，"韩渊的父亲。"

我："……"

"第一代韩家家主，在创造系统的时候，留了一个后门。"石头说，"只有真正的'理解者'，才能看到并选择 C。"

"理解者？"

"理解苏小棠的痛苦，但不被她的痛苦绑架。"石头说，"理解系统的残酷，但不被系统的规则束缚。"

"理解一切，然后……超越一切。"

我沉默了。

这话听起来……很他妈玄乎。

"那我怎么知道，我是不是'理解者'？"

"你已经是了。"石头说，"否则你根本看不到选项 C。"

我低头，看向那块石头。

"所以，"我缓缓开口，"你是在帮我？"

"我是在帮系统完成它的使命。"石头说，"系统的真正使命，不是选拔杀手，不是陪苏小棠玩，而是……"

"而是找到那个能打破循环的人。"

我深吸一口气。

"选 C，会怎样？"

"我不知道。"石头说，"但有一点可以肯定——"

"苏小棠会恨你。"

---

教室里安静得可怕。

"恨我？"我笑了，"她等了三千年的，不就是个能杀她的人吗？我选 C，不杀她，她凭什么恨我？"

"因为你剥夺了她唯一的希望。"石头说，"三千年，她唯一的执念，就是'死'。你选 C，意味着她连这个执念……都要落空。"

我沉默。

"韩林，"石头说，"你要明白一件事。"

"苏小棠不是受害者。"

"她也是加害者。"

我心头一震。

"三千年来，无数人因她而死。"石头说，"韩家九十八代人，每一代都有人死在试炼里。那些无辜的人，那些被卷入这个游戏的普通人……"

"他们的命，谁来赔？"

我没说话。

"选 A，你成为下一个苏小棠，诅咒继续。"

"选 B，系统自毁，但苏小棠继续活着，诅咒继续。"

"选 C……"石头顿了顿，"诅咒打破，但苏小棠……可能会疯。"

"疯？"

"执念落空的人，只有一个下场。"石头说，"崩溃。"

我闭上眼睛。

操。

这他妈是什么鬼选择题？

"我还有多少时间？"我问。

"六十八小时四十二分。"石头说。

"够了。"我说。

"够什么？"

"够我想清楚。"

石头没说话。

它静静地躺在课桌上，黑色的表面泛着微光。

"你……"我犹豫了一下，"你叫什么名字？"

石头沉默了很久。

"我没有名字。"它说，"但如果非要一个……"

"叫我'小黑'吧。"

我："……"

"这他妈什么破名字？"

"苏小棠以前这么叫我。"小黑的声音里，居然有一丝……怀念？

"以前？"

"在三千年前，我还是个人的时候。"小黑说，"苏小棠养过一只黑猫，她叫它'小黑'。"

"那只猫，后来死了。"

"而我……成了系统的一部分。"

我愣住了。

"你是……那只猫？"

"不完全是。"小黑说，"我只是……借用了它的记忆。"

"苏小棠创造系统的时候，把她最珍视的东西，都放了进来。"

"我是其中之一。"

我沉默了很久。

"所以，"我缓缓开口，"你其实……是站在她这边的？"

"我是站在'真相'这边的。"小黑说，"苏小棠需要知道真相，哪怕那会让她崩溃。"

"那你为什么不直接告诉她？"

"因为我不能。"小黑说，"系统的规则，禁止我直接透露。"

"但我可以……引导。"

"引导到 C 选项。"

我笑了。

"你这猫，还挺狡猾的。"

"三千年，我学会了一件事。"小黑说，"在这个系统里，想说实话，就得绕弯子。"

"直接说，会被抹除。"

我点头。

"行，我明白了。"我说，"让我想想。"

"你还有时间。"小黑说，"但记住——"

"苏小棠也在思考。"

"她不是被动等待的。"

"这六十九小时，对她来说，也是布局的时间。"

---

小黑说完这句话，就没了声音。

它重新变回了一块普通的石头，静静地躺在课桌上。

我盯着它，脑子里一团乱麻。

选 A，我成为下一个苏小棠，永生不死，永远孤独。

选 B，系统自毁，苏小棠继续活着，诅咒继续。

选 C，诅咒打破，但苏小棠可能会疯。

操。

怎么选都是坑。

"祖宗，"我忽然想起韩渊，"你在哪？"

没人回答。

我站起身，走向教室门口。

推开门。

门外是一片……虚无。

没有走廊，没有操场，没有建筑。

只有一片白茫茫的空无。

"这他妈什么鬼地方？"

我往前迈了一步。

脚下的"地面"居然是实体的。

"有意思。"

我继续往前走。

一步，两步，三步……

走了大概十分钟，前方出现了一个……人影？

我眯起眼睛。

那人影越来越近。

最后，我看清了。

是苏小棠。

她坐在一把椅子上。

一把……悬浮在虚空中的椅子。

"来了？"她笑眯眯地看着我，"比我预计的要快。"

"你他妈早就知道我会来？"

"当然。"苏小棠说，"选 A 或 B 的人，都会直接离开。只有选 C 的人……才会来找我。"

我心头一震。

"所以，"我缓缓开口，"你早就知道 C 选项的存在？"

"知道。"苏小棠说，"但我不知道，它到底是什么。"

"连你都不知道？"

"韩家祖先留的后门，我破解不了。"苏小棠说，"三千年，我试了无数种方法，都没用。"

"那你为什么还要等？"

"因为……"苏小棠歪了歪头，"我好奇啊。"

"好奇什么？"

"好奇那个能选 C 的人，到底会做出什么选择。"苏小棠说，"好奇他……会不会比我聪明。"

我笑了。

"那你觉得，我聪明吗？"

"还行。"苏小棠说，"至少比前面九十七个强。"

"那你说说，"我问，"选 C 会怎样？"

苏小棠沉默了。

很久很久。

"我不知道。"她终于开口，"但有一点可以肯定——"

"那会改变一切。"

"包括我。"

我盯着她。

"苏小棠，"我缓缓开口，"你怕吗？"

她愣了一下。

"怕什么？"

"怕未知。"我说，"你等了三千年，等一个能杀你的人。但现在，那个人可能不会杀你，而是……做一些你完全无法预测的事。"

"你不怕吗？"

苏小棠沉默了很久。

"怕。"她轻声说，"我怕得要死。"

我笑了。

"那你还等？"

"因为……"苏小棠抬起头，看着我，"这是我唯一的希望。"

"什么希望？"

"解脱的希望。"苏小棠说，"不是死亡的解脱，是……存在的解脱。"

"三千年，我一直以为，死就是解脱。"

"但现在，我不确定了。"

我沉默。

"韩林，"苏小棠忽然说，"你选 C 吧。"

我："……"

"你说什么？"

"我说，选 C。"苏小棠说，"不管那是什么，不管会发生什么……选 C。"

"为什么？"

"因为……"苏小棠笑了，那笑容里，有一丝我从未见过的东西，"我想赌一把。"

"赌什么？"

"赌你……能给我真正的解脱。"

"不是死亡，是……自由。"

---

**剩余时间：67 小时 30 分**

**选项 A：送苏小棠最后一程**

**选项 B：拒绝，等待系统自毁**

**选项 C：？？？**

**【苏小棠已表态：支持选 C】**

**【系统核心（小黑）已表态：支持选 C】**

**【韩渊：未表态】**

我盯着那个提示。

操。

连苏小棠自己都让我选 C。

这他妈……

"你怎么想的？"我问苏小棠。

"我想……"苏小棠说，"看看三千年后，这个世界会变成什么样。"

"没有我，没有系统，没有诅咒。"

"只有……自由。"

我深吸一口气。

"行。"我说，"那就选 C。"

"但在那之前……"

"我得先找到韩渊。"

苏小棠笑了。

"他就在你心里。"她说。

我愣住。

"什么意思？"

"韩渊从来就没离开过。"苏小棠说，"他只是……换了一种存在方式。"

"问你自己的心。"

"他会告诉你答案。"

我闭上眼睛。

心里，有一个声音在说话。

"小韩。"

是韩渊。

"祖宗？"

"选 C 吧。"韩渊说，"我信你。"

"为什么？"

"因为……"韩渊的声音里，有一丝笑意，"你是我韩家九十八代人里，唯一一个……让我骄傲的后代。"

我鼻子一酸。

"行了，别矫情。"我说。

"那就……动手吧。"韩渊说，"选 C，打破这个该死的循环。"

"让三千年的一切……到此为止。"

我睁开眼睛。

"苏小棠，"我说，"准备好了吗？"

她笑了。

"三千年，"她说，"我早就准备好了。"

"来吧。"

我伸出手。

按向了那个……

**【选项 C】**
