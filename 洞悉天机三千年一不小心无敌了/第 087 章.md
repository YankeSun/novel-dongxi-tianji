# 第 87 章 令牌在谁手里

韩渊盯着那个被定格的男人。

天字一号试炼者。

或者说，另一个版本的自己。

现在这人像一尊雕塑，保持着说话的姿势，嘴巴微张，眼神凝固在某个瞬间。

挺他妈滑稽的。

「你把他怎么了？」苏婉清的声音有点抖。

「没怎么。」韩渊收回手，「就是让他安静一会儿。」

时间开始流动。

天字一号试炼者猛地吸了一口气，像是刚从水里钻出来。

「你——」他盯着韩渊，眼神里有种……恐惧？

有意思。

刚才还一副「老子要成神了」的架势，现在怂了？

「你到底是什么？」天字一号试炼者问。

「我？」韩渊指了指自己，「我就是韩渊啊。你不是刚说过吗，天字七号试炼者，失败品，三千年才完成融合的废物。」

天字一号试炼者不说话了。

他往后退了一步。

这个动作很微妙。

不是战术性的后退，是本能性的。就像人看见蛇会后退一样，不管那蛇有没有毒。

「你身上有天机的味道。」天字一号试炼者说。

「哦？」韩渊笑了，「那你闻闻，像不像？」

天字一号试炼者脸色变了。

他当然能闻到。

那股味道，就像……就像一个人身上同时有生者和死者的气息。矛盾，混乱，但又诡异地和谐。

「你融合了多少？」他问。

「不知道。」韩渊耸耸肩，「没数过。」

「你在撒谎。」

「对。」韩渊点头，「我在撒谎。但你能拿我怎么样？」

天字一号试炼者沉默了。

他能怎么样？

刚才那一瞬间，他连自己怎么被定格的都没反应过来。这种能力，已经超出了修行者的范畴。

这是……规则层面的东西。

「你走吧。」韩渊说。

天字一号试炼者愣住了。

「什么？」

「我说你走吧。」韩渊挥挥手，「趁我还没改变主意。」

「你……」天字一号试炼者盯着他，像是在判断这是不是陷阱。

「别误会。」韩渊说，「我不是放你走。我是觉得，杀你太浪费时间了。」

天字一号试炼者的脸抽了抽。

这话比直接说「我要杀你」还侮辱人。

「你会后悔的。」他说，「其他五个修行者不会放过你。」

「让他们来。」韩渊说，「正好我最近有点闲。」

天字一号试炼者又沉默了三秒。

然后他转身，走向传送阵。

走到传送阵门口时，他停住了。

「有件事，你应该知道。」

「说。」

「令牌。」天字一号试炼者说，「七块令牌，已经有三块现世了。」

韩渊的眼神冷了下来。

「哪三块？」

「第一块，在天机手里。」天字一号试炼者说，「第二块，在苏婉清手里。第三块——」

他顿了顿。

「在你脑子里。」

传送阵的门关上了。

韩渊站在原地，没动。

苏婉清也没动。

空气安静得能听见灰尘落地的声音。

「解释一下。」韩渊说。

苏婉清深吸一口气。

「没什么好解释的。」她说，「他说的对。」

「你手里有令牌？」

「有。」

「什么时候的事？」

「很久以前。」苏婉清说，「久到你还没分裂的时候。」

韩渊笑了。

笑得很他妈难看。

「所以。」他说，「你一直都知道。」

「知道什么？」

「知道我是谁，知道天机是谁，知道这整个该死的计划。」韩渊一步步走向她，「而你，一直在旁边看着。像看戏一样。」

苏婉清没躲。

她迎着韩渊的目光，眼神里有种……疲惫？

「我不是在看戏。」她说，「我是在等。」

「等什么？」

「等你想起来。」

韩渊停住了。

「想起来什么？」

苏婉清从储物袋中掏出一样东西。

那是一块令牌。

不是普通的令牌，是那种……看起来就很古老的令牌。表面有复杂的纹路，像是某种符文，又像是某种文字。

令牌在发光。

很微弱的光，但确实存在。

「这是第二块令牌。」苏婉清说，「对应的是『记忆』。」

韩渊盯着那块令牌。

他感觉脑子里有什么东西在跳动。

像是……被触发了。

「我说过，我是天道监察使。」苏婉清继续说，「但我没说的是，我为何监察。」

「为什么？」

「因为三千年前，你求我这么做。」

韩渊愣住了。

「我……求你？」

「对。」苏婉清点头，「三千年前，你发现天机的计划之后，知道自己阻止不了。所以你做了两件事。」

「哪两件？」

「第一，你把自己分裂成两半。一半留在人间，一半成为天机。」苏婉清说，「第二，你找到我，把第二块令牌交给我，让我等你。」

「等我什么？」

「等你重新融合的那一刻。」苏婉清说，「等你需要这块令牌的时候。」

韩渊不说话了。

他感觉脑子里乱成一团。

三千年的记忆，像潮水一样涌上来。但又模糊，像隔着一层雾。

「我……」他张了张嘴，「我为什么要这么做？」

「因为你发现了一个秘密。」苏婉清说，「一个连天机都不知道的秘密。」

「什么秘密？」

苏婉清看着他。

眼神很复杂。

有同情，有无奈，还有……某种韩渊读不懂的东西。

「七道令牌不是大逃杀。」她说，「是献祭。」

韩渊感觉心脏停了一拍。

「献祭？」

「对。」苏婉清说，「七个修行者，七个祭品。召唤的不是神，是牢笼。」

「牢笼？」

「天机把自己锁在里面了。」苏婉清说，「三千年前，他——或者说你，发现了某个东西。某个不能让世人知道的东西。」

「什么东西？」

「我不知道。」苏婉清摇头，「你没告诉我。你说，等我见到你的时候，我自然会知道。」

韩渊沉默了。

他忽然觉得，自己像个傻子。

被自己耍了三千年。

「所以。」他说，「现在的局面是——天机要集齐七块令牌，打开牢笼。而我——」

「你要阻止他。」苏婉清说，「或者说，阻止你们。」

「怎么阻止？」

「找到剩下的令牌。」苏婉清说，「在他之前。」

韩渊笑了。

「说得轻巧。七块令牌，三块现世，剩下四块在哪？」

「我不知道。」苏婉清说，「但我知道，有人正在找。」

「谁？」

「所有修行者。」苏婉清说，「你以为只有天字一号试炼者在找你？其他五个，都在。」

韩渊不说话了。

他走到窗边，看着外面的京城。

凌晨的天空，灰蒙蒙的。

那道裂缝还在。

横跨整个天空，像一道伤口。

「他们什么时候到？」他问。

「不知道。」苏婉清说，「但不会太久。」

韩渊点点头。

然后他转过身。

「有件事，我需要确认。」

「什么？」

「艾薇。」韩渊说，「她到底是不是——」

他没说下去。

但苏婉清明白了。

「她是你的分身。」苏婉清说，「但不是你理解的那种分身。」

「什么意思？」

「她是你的……心魔。」苏婉清说，「三千年前，你分裂自己的时候，把一部分人性剥离出来，化作分身投射到了现实世界。」

韩渊感觉脑子里有什么东西炸开了。

「所以……」

「所以她没有母亲。」苏婉清说，「她是凭空出现的。七年前，在西域的一座城池中。有人说她早产，母亲大出血死了。但那是假的。」

韩渊沉默了。

他想起艾薇。

想起她小时候的样子，想起她第一次叫他爸爸时的表情，想起她生病时他守在她床边的每一个夜晚。

原来都是假的。

或者说，不完全是假的。

「她知道吗？」他问。

「不知道。」苏婉清说，「天机也不知道。只有我知道。」

「为什么告诉我？」

「因为你需要这个。」苏婉清说，「当你面对他的时候，你需要知道，你守护的是什么。」

韩渊笑了。

笑得很他妈凄凉。

「所以这是个陷阱。」他说，「从一开始就是。天机要的不是令牌，是我。他要我自愿走进祭炼大阵，完成融合。」

「对。」

「那如果我拒绝呢？」

「艾薇会死。」苏婉清说，「不是比喻，是真的会死。她是你的分身，你拒绝融合，她就没有存在的意义。」

韩渊不说话了。

他走到蒲团边，坐下。

然后他捂住脸。

很久。

「我有的选吗？」他终于说。

「有。」苏婉清说，「你可以掀桌子。」

韩渊抬起头。

「掀桌子？」

「对。」苏婉清说，「不按照他的规则玩。不按他的剧本走。」

「怎么掀？」

「我不知道。」苏婉清说，「但我知道，你一定会想到办法的。」

韩渊看着她。

看了很久。

「你对我很有信心。」他说。

「不是信心。」苏婉清说，「是了解。」

「了解什么？」

「了解三千年前的那个你。」苏婉清说，「他是个疯子。但也是个天才。他能在绝境中找到出路，能在死局里下出活棋。」

韩渊笑了。

「那他现在呢？」

「现在？」苏婉清也笑了，「现在他多了三千年的经验。应该更疯了吧。」

韩渊不说话了。

他站起来，走到窗边。

外面的天，开始亮了。

第一缕阳光照进房间，落在地板上。

像一道光。

「苏婉清。」他说。

「嗯？」

「如果有一天，我需要你帮我做一件事。」韩渊说，「一件可能让你死的事。」

「我会做。」苏婉清说。

「不问为什么？」

「不问。」

韩渊点点头。

「好。」

他转过身。

「那我们现在去哪？」苏婉清问。

「西域。」韩渊说，「去接我分身回来。」

「天机会在那里等你。」

「我知道。」

「那是陷阱。」

「我知道。」

「那你还要去？」

韩渊笑了。

笑得很他妈好看。

「陷阱怎么了？」他说，「我韩渊这辈子，就是在陷阱里长大的。」

他走向门口。

「走吧。」

苏婉清跟上去。

走到门口时，她停住了。

「韩渊。」

「嗯？」

「有件事，我刚才忘了说。」

「什么？」

苏婉清看着他。

眼神里有种……韩渊读不懂的东西。

「第三块令牌，在你脑子里。」她说，「但它不是完整的。」

「什么意思？」

「它需要被激活。」苏婉清说，「而激活它的方法——」

她停住了。

「是什么？」韩渊问。

「是让你在乎的人，在你面前死一次。」

韩渊愣住了。

然后他笑了。

笑得很他妈难看。

「所以。」他说，「这是个死局。」

「不是。」苏婉清摇头，「死局是你没得选。但这个局，你有选择。」

「选什么？」

「选谁死。」苏婉清说，「艾薇，还是我。」

韩渊盯着她。

很久。

「我选第三个。」他说。

「什么？」

「我选掀桌子。」韩渊说，「不让任何人死。」

苏婉清沉默了。

三秒后，她笑了。

「那就试试看吧。」她说，「看看你能不能，真的掀了这张桌子。」

韩渊没说话。

他打开门，走出去。

长廊里很安静。

安静得能听见自己的心跳。

一下，一下，一下。

像是在倒计时。
